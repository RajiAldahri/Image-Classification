# Image Classification

The code implements a Convolutional Neural Network (CNN) using TensorFlow and Keras for image classification on the CIFAR-10 dataset. Here's a short description of each key section:

**1- Data Loading and Preprocessing:**

CIFAR-10 dataset is loaded, containing 60,000 32x32 color images in 10 different classes.
Pixel values are normalized to be in the range [0, 1].
Class labels are one-hot encoded.

**2- Data Splitting:**

The training data is split into training and validation sets using a 80-20 split ratio.

**3- CNN Model Definition:**

A Sequential model is created representing a deep CNN architecture.
It consists of multiple convolutional layers with max-pooling, dropout for regularization, and fully connected layers.
The output layer has softmax activation for multi-class classification.

**4- ImageDataGenerator for Augmentation:**

An ImageDataGenerator is configured for real-time data augmentation.
Augmentation includes random rotations, horizontal shifts, and flips to enhance the model's generalization.

**5- Model Compilation:**

The CNN model is compiled with categorical cross-entropy loss and the Adam optimizer with a specified learning rate and decay.

**6- Model Training:**

The model is trained using the augmented data generated by ImageDataGenerator.
Training is performed for 100 epochs with a batch size of 64.
Validation data is used to monitor the model's performance during training.

The overall goal is to train a robust image classification model on CIFAR-10, leveraging data augmentation to improve the model's ability to generalize to unseen data. The architecture includes convolutional and fully connected layers with regularization techniques to prevent overfitting.

# Overview

My trained Convolutional Neural Network (CNN) on the CIFAR-10 dataset achieved an accuracy of 86% :shipit:. This indicates the successful learning of intricate patterns in the images and effective generalization to previously unseen data. The model, configured with data augmentation for enhanced robustness, showcases strong performance in image classification tasks.
